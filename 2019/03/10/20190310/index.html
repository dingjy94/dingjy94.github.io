整体平平无奇的一周。

捡了捡python，完善之前的React项目，算是好好学了一下怎么写测试。断断续续刷了些题。

锻炼倒是一直keep，应该是来美国之后坚持最好的一次了，果然花钱就是不一样。

书的话主要是继续看clean code 和 you don’t know JS，顺便吞了些看起来比较有意思的技术书籍在微信阅读上。

周末一顿狂吃，久违的吃了寿喜烧可丽饼，刚刚又去吃了附近的那家心心念很久了的甜品店，感觉一周白锻炼了 (捂脸。

还看了两场电影Madea的新系列和Apollo11。看这类黑人喜剧片以我的听力和笑点还是有点吃力，倒是最后那段关于家庭婚姻的讨论感触颇深。

Apollo11倒是真的很不错，第一次在大银屏上看纪录片，而且和别的纪录片不同甚至都没有什么旁白，就是单纯的把NASA原始资料和新闻录像拼接到了一起，但是这种纯粹的真实却感觉比完整的电影更加的震撼。

看完之后一直在想的是整整50年过去了，当时的科技和工业完成了这样的伟业，那这五十年我们究竟有什么进步呢。信息产业发展，第三次工业革命，互联网革命，到现在的人工智能热潮，这五十年尤其是这20年，让我们体感最深的科技发展几乎围绕着EECS。

作为一个混这口饭也很喜欢cs和coding的人倒是并不想黑cs，而且我坚持认为cs或者信息产业绝对不是什么“虚”或者“软”的东西，也不是纯粹的第三产业。信息革命给传统工业和管理带来的效率提高肉眼可见，对科研带来的辅助作用也是不小，对普通百姓生活水平带来的提升更是瞎眼可见。

但是cs，或者说cs优秀的商业化能力——特别是互联网方面，带来的潜在的问题就是过于多的人力和资本投入到了这个领域。然而cs毕竟某种意义上确实是一个辅助学科——他的意义在于对传统行业带来效率上的绝对提升——但如果传统工业和基础学科本身做不出突破的话cs并不能孕育出颠覆性的东西——当然如果人工智能实现的话是有这个希望。

这个问题存在于美国，但是最严重的是在国内。美国虽然湾区热闹非凡，但传统工业也没有完全没落，科研方面也是有无数人才愿意奉献终身。但是国内这些年来的风气，有点太过于cs至上了——而且传统工业没落的太快了。大概也有一部分改革开放以来实用主义至上的风气带来的影响。

当然作为一个码农说这些有点可笑，我自己终究也是毫不犹豫顺着这个风潮走的小人物。而且我也并没觉得在这个年代动用举国体制实现这种经济上收益甚微的伟业值得过分推崇。现在的美国不会再做这种事情，我国自改革开放以来就不会做这种事情，战斗民族现在显然有心也无力。

只是看完这个纪录片，对当年那些科研工程人员感到了钦佩，也想起了我航那些投身航空航天事业的老教授们，校友们，同学们。我一直都不太喜欢我航的死板氛围家国情怀，但还是钦佩这些真的能抱着“航天梦”投身航空航天事业的人。

“我们的征途是星辰大海”，现在的我对这句话已经没什么感觉，但是我真的佩服和羡慕把这句话当做信条的人们。

另外Apollo11用的音乐我都很喜欢，其中John Stewart的Mother Country很是符合这个片子的背景。”Just a lot of people doing the best they could”，这是所有奇迹的根源。
